1. TikaDocumentReader核心作用

  • 自动识别文档格式：无需手动指定文档类型（如 PDF、Docx），它会自动检测文件格式并选择对应的解析器；
  • 提取纯文本内容：忽略文档的格式、样式、图片等非文本信息，只提取核心文字内容，供后续嵌入模型生成向量；
  • 封装为 Spring AI 标准 **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Document</font>** 对象：提取的文本会被封装为 Spring AI 统一的 <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Document</font> 类型（包含文本内容和文档元数据），直接对接后续的向量存储组件，无需手动转换格式。
